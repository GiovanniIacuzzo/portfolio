3D MRI Image Registration using CPSO

This project focuses on the alignment of three-dimensional MRI volumes using the CPSO algorithm. The goal is to improve image quality and diagnostic accuracy while reducing artifacts and discrepancies between successive scans.

The solution combines deep learning methods with classical image registration techniques, enabling accurate mapping between different scans. This approach is useful both in clinical settings and neuroscience research.
